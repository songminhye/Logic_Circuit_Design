# Logic_Circuit_Design



## [주어진 필수 조건]

### Xilinx Virtex4 프로그램으로 자판기 제작 
 
 최종금액 : 
 
                    650원 
 
 사용주화 : 
 
                   50원, 100원, 500원
 
 필수버튼 (스위치) : 
 
                    c50, c100 c500 (입력)
 
                    reset
                    
                    coffee (출력)
                    
                    
                    
                    

## [만든 결과]

  입력 포트 : 
  
             c50 : 50원을 주입
  
             c100 : 100원을 주입
             
             c500 : 500원을 주입
             
             Bcoffee : 커피 뽑는 버튼
             
  출력 포트 : LED_coffee : 커피를 뽑을 수 있을 때 불 들어오는 신호
  
             r50 : 50원 반환
             
             r100 : 100원 반환
             
             r500 : 500원 반환
             
             Gcoffee : 커피가 나오는 신호 
             
             
             
  ## <세부기능>
  
    1) Gcoffee : 300원이 되면 커피를 뽑는다. 
    
    2) 650원을 초과해서 돈을 넣을 경우, 넣은 돈을 바로 반환한다.
    
    3) 동전 상태가 300원 이상이 되면 LED_coffee에 신호가 들어온다. 
    
    4) reset 기능
